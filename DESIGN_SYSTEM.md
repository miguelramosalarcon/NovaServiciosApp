# 🎨 Nova Servicios App – Design System

Este documento define el sistema visual y lineamientos de diseño utilizados en la aplicación Nova Servicios App.

El objetivo es mantener coherencia visual, profesionalismo y escalabilidad en el desarrollo.

# 🖋 2. Sistema Tipográfico

Tipografía base: **Roboto** (por defecto en Android)

## Escala Tipográfica

| Elemento | Tamaño | Peso |
|----------|--------|------|
| Título principal | 24sp | Bold |
| Subtítulo | 18sp | SemiBold |
| Texto normal | 16sp | Regular |
| Texto secundario | 14sp | Regular |
| Texto botones | 16sp | Medium |

---

# 🔘 3. Botones

## 🔵 Botón Primario

- Background: #3A7BD5
- Texto: #FFFFFF
- Radius: 12dp
- Padding vertical: 12dp

| Estado | Color |
|--------|--------|

## 🟢 Botón Secundario

# 📏 4. Sistema de Espaciado

Se utiliza la regla de múltiplos de 8 para mantener consistencia visual:

- 8dp
- 16dp
- 24dp
- 32dp

Nunca usar valores arbitrarios.

---

# 📱 5. Cards

- Background: #FFFFFF
- Border Radius: 16dp
- Elevation: 4dp
- Margen externo: 16dp
- Padding interno: 16dp

# 🎯 7. Principios UI Aplicados

- Minimalismo
- Jerarquía visual clara
- Uso correcto del color para acciones
- Consistencia en espaciado
- Material Design como base
- Experiencia centrada en el usuario
