# 📱 Nova Servicios App

Aplicación móvil desarrollada en **Java** con **Android Studio**, como proyecto académico del curso:

## PDSD-641-647 – Desarrollo de Aplicaciones Móviles

Este proyecto simula el desarrollo de una app corporativa para la empresa ficticia **Servicios Integrales Nova**, dedicada a brindar soluciones técnicas a domicilio (plomería, electricidad y mantenimiento).

---

# 🎯 Propósito Académico

El objetivo de este repositorio es documentar el desarrollo progresivo de una aplicación Android siguiendo la guía práctica del curso, aplicando estándares profesionales desde la etapa formativa.

Se trabaja bajo una estructura por fases:

- HT01 – Configuración del entorno y estructura base
- HT02 – Diseño de interfaces y eventos
- HT03 – Navegación y almacenamiento local
- HT04 – Consumo de API REST
- HT05 – Publicación y versionado

---

# 🧭 Fases del Proyecto

## 🔵 HT01 – Configuración Inicial

- Instalación y configuración de Android Studio
- Configuración del SDK
- Creación del proyecto en Java
- Exploración de la estructura base

✔ Resultado: Proyecto base funcional y correctamente estructurado.

---

## 🟣 HT02 – Diseño de Interfaces (UI + Eventos)

**Objetivo:** Diseñar la pantalla principal utilizando XML y programar eventos en Java.

Incluye:

- Creación de layouts XML
- Implementación de controles (Button, TextView)
- Programación de eventos
- Aplicación de principios básicos de Material Design

✔ Resultado: Pantalla principal moderna, clara y funcional.

---

## 🟢 HT03 – Navegación y Base de Datos Local

**Objetivo:** Implementar múltiples pantallas y almacenamiento con SQLite.

Incluye:

- Creación de múltiples Activities
- Uso de Intents para navegación
- Base de datos SQLite
- Operaciones CRUD en Java

✔ Resultado: Aplicación capaz de registrar y consultar servicios.

---

## 🟡 HT04 – Consumo de Servicios Web

**Objetivo:** Conectar la aplicación a una API REST y mostrar datos remotos.

Incluye:

- Conexión HTTP
- Parseo de JSON
- Manejo de errores de red
- Visualización en ListView o RecyclerView

✔ Resultado: Aplicación sincronizada con datos externos.

---

## 🔴 HT05 – Publicación en Google Play

**Objetivo:** Preparar, firmar y publicar la aplicación.

Incluye:

- Generación y firma de APK
- Configuración de versión y permisos
- Publicación en Google Play Console
- Simulación de actualización

✔ Resultado: Aplicación lista para entorno real.
